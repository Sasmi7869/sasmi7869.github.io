## ✅ Verifying Backup Integrity

Verification ensures your files remain readable and complete. Perform this check after significant device updates:

1.  Go to the Backup section of the main menu.
2.  Select the backup you want to check.
3.  Click the Verify button.
4.  The application compares the data on the device to the data in your backup folder.
5.  Wait for the result. The application shows a green checkmark if the files are correct.

If the software reports an error, create a new backup immediately.

## 🔎 Frequently Asked Questions

**Does this software store my backup password?**
No. The application does not store your backup password. You must manage this password yourself. Lost passwords mean that you cannot recover your backup data.

**Can I move the backup location?**
Yes. Open the settings menu within the application. Select the Storage tab. You can choose any folder on your computer or an external hard drive as the target path for your backups.

**My device does not appear in the menu.**
Ensure your cable works. Try a different USB port on your computer. Check that your computer recognizes the iPhone in your file explorer. Restart both the iPhone and the computer if the issue continues.

**Will this software slow down my computer?**
The application uses system resources only when performing backups or verification tasks. You can minimize the window to the tray to keep your workspace clear.

**Do I need an internet connection?**
An internet connection is not required to create or verify backups. You only need the internet to download the original application installer.
